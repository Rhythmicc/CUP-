# CUP_EXAM  

## Environment dependence:  
`pip3 install requests`

`
pip3 install xlrd
`

## Usage:  
`python3 main.py`  
or  
`chmod 777 ./install.sh`  
`./install.sh`   
will pack an  executable program. Then execute:
`./main`
