# CUP_EXAM  

## Environment dependence:  
```shell
pip3 install requests
pip3 install xlrd
```

## Usage:  
```shell
python3 frontend.py 
```

- If your OS is Mac OS or Linux, then you can run :

- ```shell
  ./install(<mac>/<linux>).sh
  ```

  and it will create a command 'exam'. 

  After that, you can run:

  ```shell
  exam
  ```

  to use program~
